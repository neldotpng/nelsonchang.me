Designation Showcase

UI Design / Front-End

Wanderlust Trunk Club

I worked with another front end developer to design and develop a showcase for the current students and previous graduates of Designation. We were given two weeks to design and develop the showcase.

Skills

HTML/CSS, SASS, JSON, Ajax, JavaScript, jQuery, Responsive

Deliverables

Design Mockups, High Fidelity Prototype

Design Process

Given the short time frame for the project, we started sketching and iterating designs immediately. We were given constraints for features that needed to be included on the webpage. Apart from what student information needed to be displayed, the showcase needed to also be filterable and searchable.

We decided on a card layout featuring the ability to expand cards and display more information on a given student.

Based on my initial sketches, I started iterating on card designs, expanded card views, and page layout schemes in Illustrator.

Final Design

Because of the timeframe of the project, I got the designs to about 90% fidelity and had them approved by our client so that my teammate and I could start work on coding the site.

Development

Isotope.js + Angular.js

My teammate worked with Isotope.js for filtering and with Angular.js for populating the page content.

HTML + CSS

I coded the HTML from scratch and used SASS to style the page. Since I did not create official mobile and tablet designs, I created the responsive designs as I coded the CSS.

JavaScript + jQuery

Lastly, I worked with JavaScript and jQuery to code in animations and join the filter functionality with my teammate's Isotope code.
